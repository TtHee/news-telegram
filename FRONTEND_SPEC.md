# 前端設計規格報告
## 個人新聞智能推播平台 — docs/index.html

---

## 一、技術限制

- **單一靜態 HTML 檔案**，部署於 GitHub Pages（`docs/index.html`）
- 不可有後端、不可有 Node.js，只能用原生 HTML + CSS + JavaScript
- 資料來源：同目錄下的 `data/news.json`（用 `fetch()` 讀取）
- 需支援手機瀏覽器（使用者主要用手機看）

## 三、各區塊功能與欄位說明

### 3.1 頂部 Header

| 元素 | 說明 |
|------|------|
| 平台名稱 | 固定文字，如「每日智聞」 |
| 搜尋框 | 即時過濾新聞標題與摘要，輸入即篩選，不需按鈕 |
| 最後更新時間 | 從 `news.json` 的 `generated_at` 欄位顯示，格式：`更新於 03/14 09:15` |

---

### 3.2 左側欄（桌機版固定，手機版為 Tab）

**① 預警燈號區塊**

```
┌─────────────────────┐
│  系統風險指數        │
│  ████████░░  72/100 │
│  🔴 高風險留意       │
│                     │
│  ⚠️ VIX 高位 28.3   │
│  ⚠️ 台股跌 -3.2%    │
└─────────────────────┘
```

| 欄位 | 資料來源 | 說明 |
|------|---------|------|
| 分數 | `risk_score` | 0～100 整數 |
| 進度條顏色 | `risk_level` | normal=綠、watch=黃、danger=紅 |
| 燈號文字 | `risk_level` | 🟢 正常 / 🟡 留意 / 🔴 高風險 |
| 警示清單 | `risk_signals[]` | 最多顯示 3 條 |

**② 市場數據區塊**

```
┌─────────────────────┐
│  📊 市場概況         │
│  台股    19,850      │
│          ▼ -0.8%    │
│  S&P500  5,200      │
│          ▲ +0.3%    │
│  VIX     18.3       │
│  美債10Y  4.25%     │
└─────────────────────┘
```

| 欄位 | 資料來源 | 說明 |
|------|---------|------|
| 台股 | `market.TWII` | price + change_pct，漲紅跌綠 |
| S&P 500 | `market.SP500` | 同上 |
| VIX | `market.VIX` | 只顯示價格，>25 標紅 |
| 美債10Y | `market.TNX` | 只顯示殖利率 |

**③ 分類選單**

點擊切換右側顯示的新聞分類：
- 全部
- 🤖 AI
- 🌍 全球趨勢
- 💰 財經
- 📉 台股
- 📈 美股

另有篩選按鈕：
- `全部` / `未讀` / `已收藏`

### 3.3 右側新聞卡片

每張卡片包含以下欄位：

```
┌─────────────────────────────────────┐
│ ✅ 正面  📉 台股          [⭐] [✓]  │  ← 情緒標籤、分類標籤、收藏/已讀按鈕
│                                     │
│ Fed 宣布維持利率不變，市場反應平靜    │  ← 標題（粗體，最多2行）
│                                     │
│ 聯準會本次會議決定維持基準利率在      │  ← AI 摘要（最多3行，超過省略）
│ 5.25%～5.5% 區間，主席鮑爾表示...   │
│                                     │
│ Reuters  •  2小時前        [閱讀原文]│  ← 來源、時間、連結
└─────────────────────────────────────┘
```

| 欄位 | 資料來源 | 說明 |
|------|---------|------|
| 情緒標籤 | `sentiment` | 正面=✅綠色、負面=❌紅色、中性=▫️灰色 |
| 分類標籤 | `category` | 小標籤顯示分類名稱 |
| 重大標記 | `is_breaking` | 顯示 🔔 Breaking 紅色標籤 |
| 標題 | `title` | 粗體，最多 2 行截斷 |
| AI 摘要 | `summary_zh` | 最多 3 行，可點擊展開全文 |
| 來源 | `source` | 灰色小字 |
| 時間 | `published_at` | 轉換為「X 小時前」或「X 分鐘前」 |
| 閱讀原文 | `url` | 新分頁開啟 |
| 收藏按鈕 | `saved` | ⭐ 點擊切換，狀態存 localStorage |
| 已讀按鈕 | `read` | ✓ 點擊切換，已讀卡片變淡，狀態存 localStorage |

## 四、互動功能

| 功能 | 行為 |
|------|------|
| 關鍵字搜尋 | 即時過濾標題 + 摘要，不分大小寫 |
| 分類切換 | 點擊左側分類，右側只顯示該分類卡片 |
| 未讀篩選 | 只顯示 `read=false` 的卡片 |
| 收藏篩選 | 只顯示 `saved=true` 的卡片 |
| 收藏/已讀 | 點擊後狀態存入 `localStorage`，重整頁面不消失 |
| 摘要展開 | 點擊卡片可展開完整摘要 |
| 重大新聞置頂 | `is_breaking=true` 的卡片優先排列在最上方 |

## 七、給 AI 設計師的實作提示

1. **不需要框架**，純 HTML + CSS + JS 即可，避免引入 React/Vue 等
2. **收藏與已讀狀態**用 `localStorage` 儲存，key 為 `news_saved` 和 `news_read`，value 為 ID 陣列的 JSON
3. **時間轉換**：`published_at` 為 ISO 8601 格式，請轉為「X 分鐘前 / X 小時前 / X 天前」
4. **資料載入**：用 `fetch('./data/news.json')` 讀取，若失敗顯示友善錯誤訊息
5. **手機優先**：左側欄在手機版改為頂部水平捲動 Tab，卡片改為單欄
6. **重大新聞**：`is_breaking: true` 的卡片加上紅色 `🔔 Breaking` badge 並置頂
7. **進度條**：風險指數用 CSS `width` 動態設定，顏色依 `risk_level` 切換
8. **無資料處理**：`change_pct` 可能為 `null`，市場數據 `price` 可能為 `null`，需做防呆
